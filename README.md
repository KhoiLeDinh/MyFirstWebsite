# MyFirstWebsite
This is my first website designed, for a school project, using HTML, CSS, JS &amp; Python (Flask)
📖 Poetry Website Project
📌 Overview

This project is a simple full-stack web application developed as part of a high school IT course. It presents a multilingual poem along with author information, historical context, and a contact feature.

The website is built using HTML, CSS, JavaScript, and Python (Flask), demonstrating basic frontend design and backend functionality.

🎯 Features
📜 Display poem in multiple languages (French, English, Vietnamese)
👤 Author information section
🧭 Navigation system using dynamic section switching
📝 Contact form with backend handling
🖼️ Image integration for visual presentation
🎨 Styled layout using CSS
🛠️ Tech Stack
- Frontend
- HTML5
- CSS3
- JavaScript
- Backend
+ Python
+ Flask
+ Database (optional / future improvement)
+ SQLite
